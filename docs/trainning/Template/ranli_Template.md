# 完整公告

## 作者简介
陈楸帆，1981年生于广东汕头，毕业于北京大学中文系中国语言文学专业、艺术学院影视编导专业双学位，现任中国作家协会科幻文学委员会副主任、世界华人科幻协会会长。他是中国当代最活跃的科幻作家之一，作品被翻译为多国语言，曾多次获中国科幻银河奖、全球华语科幻星云奖等重要奖项。他的写作以现实主义与科幻想象的巧妙融合而著称。

## 作品简介
《荒潮》是陈楸帆最具代表性的长篇小说，自出版以来广受好评，被誉为“中国近未来科幻的里程碑之作”。

小说以近未来的电子垃圾回收岛“硅屿”为背景，讲述了外来女孩小米在卷入本地家族势力、国际资本与技术黑潮的冲突中，逐渐觉醒并反抗的故事。作品深刻探讨了技术伦理、阶级分化、环境危机与人性异化等议题。

该书曾获华语科幻星云奖长篇小说金奖。
