## 部长：活动主要负责人/总指挥
1. 从经过内训的干员中招募活动项目组的人手
2. 确定活动的举办日期
3. 在社长放鸽子时，承担活动的必要流程，这些流程包括
	1. 素质拓展网，活动立项
	2. 素质拓展网，导入二课分加分名单
	3. 团在浙大，学生活动申请
	4. 团在浙大，活动备案
	5. 团在浙大，基金项目申请审批流程
	6. 报销流程
	7. 教室借用
4. 收集活动的照片，用于纪念、写推文、跑流程
5. 收集活动的报销材料
6. 监督活动进程，分派举办活动所需的任务
