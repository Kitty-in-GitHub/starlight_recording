***至少是免费的***

最后更新日期 2026/1/23

天天开心



# 第一步：登录和选择
1. 登录这个网站 [教室资源管理中心](http://jxzygl.zju.edu.cn/)
2. 点这个”教室借用“![500](img/Pasted%20image%2020260123162252.png)
3. 选择”普通借用“
4. 统一身份认证登录（如果你近期在其他网站使用了统一身份认证登录，则可能会自动登录）
5. 弹出“借用须知”界面，点”已阅读“
6. 此界面，左侧”借用方式“默认选择”按日期节次借用“，左侧信息一般只需要注意”教学楼“，”起止时间“和”节次“![500](img/Pasted%20image%2020260123162921.png)
7. 点击查询后，会在下方显示可以借用的教室。找到了你想借用的教室点击选中你想借的教室（注意不要点击查看详情，这个按钮可以看到教室的详细信息，但需要再填一遍借用的日期节次），然后点击”借用“按钮
8. 弹出的界面按照界面提示填写即可。”借用原因“选择社团。
9. 点击”下一步“之后你还是要继续填表

# 第二步：填第二张表

- 借用人信息
	- 单位需要改成文学院，单位需要改成文学院，单位需要改成文学院。
	- 其余信息自行填写即可，通常会自动填充。
	- **单位一定要改成文学院**
- 活动信息
	- 活动负责人：输入”金进“，选择文学院的那个。所在单位”文学院“，其余内容参见[金进中文主页-浙江大学个人主页](https://person.zju.edu.cn/0015055)
	- 活动组织者：填你自己，单位选文学院
	- 指导老师：金进
	- 四道单选题全部选”否“

提交即可，别忘了过几天看看教室审核有无通过（钉钉信息，或通过此界面的”借用记录“查看）![500](img/Pasted%20image%2020260123162921.png)
