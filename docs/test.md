## 字号和字色
markdown没有自己调节字号的方法，只能通过嵌入html语法改变字号字色
```
<font color=red> 改变字色为红色
<font color=#00FF00> 改变字色为绿色
```

<font color=red>改变字色为红色 
<font color=#00FF00>改变字色为绿色<font color=black>
